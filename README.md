# mk_example_enterprise_m2

## Installation and setup

**Dependencies:**

- python

### mkdocs

```bash
pip install mkdocs
```

### mkdocs-static-i18n
```bash
pip install mkdocs-static-i18n
```

### mkdocs-material

```bash
pip install mkdocs-material
```

## Run

```bash
python -m mkdocs serve
```

## Deploy

```bash
pythom -m mkdocs build --no-directory-urls
```
